# 관리자 · 원시자료 화면 구성 목적

## 목적

**재직 사원** 목록에서 **한 행(한 사원)**을 선택하면, 지정한 **조회 기간**에 해당하는 그 사원의 **출근·퇴근 원시 기록**이 **오른쪽 원시자료** 테이블에 표시되는 것이 이 화면의 목적이다.

## 역할 분담

| 영역 | 역할 |
|------|------|
| 상단 필터 (성명, 시작일, 종료일) | **조회 기간** 및 필요 시 성명 검색 조건을 정한다. |
| 왼쪽 · 재직 사원 테이블 | **누구의** 기록을 볼지 선택한다 (행 선택). |
| 오른쪽 · 원시자료 테이블 | 선택한 사원 + 조회 기간에 맞는 **출퇴근 기록**을 보여 준다. |

정리하면, 왼쪽은 **주체(사원)** 선택, 상단은 **기간(및 검색)** 조건, 오른쪽은 **그 조합에 대한 결과**이다.

## 레이아웃 관련 메모 (의도)

- 재직 사원 영역의 가로 크기를, 상단 줄에서 **종료일 날짜 입력창의 오른쪽 테두리**와 세로로 맞추는 구성을 의도할 수 있다 (시각적 기준 정렬).

## 현재 구현 상태

- UI 목업(테이블·필터·일자 열 요일 색 등)은 있으며, **행 선택 시 오른쪽 테이블을 채우는 연동**은 API·상태 연결 시 구현한다.
